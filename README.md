
# Ajax
전체 페이지를 다시 불러오지 않고 웹서버에 정보를 요청해 부분적으로 정보를 갱신

```
fetch API 기본 사용법

fetch('html').then(function(response){
        response.text().then(function(text){
          alert(text);
        })
})
폴더 내에 html 파일을 찾고(fetch())
그 결과를 받아서(response)
파일 내용(text)을 경고창으로 출력(alert())


```
